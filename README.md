# Sistema GPS de Passagens - Advertorial + Mini Blogs

Projeto completo de advertorials e mini blogs para promover o Sistema GPS de Passagens da Lari Colares.

## Estrutura do Projeto

### 4 Entregas Principais:

1. **Advertorial Principal** (página única)
   - Estética de blog de notícia
   - Imagens da Lari viajando
   - Prints comparativos Google Flights vs Sistema GPS
   - CTA forte para o Sistema GPS

2. **Mini Blog Principal**
   - Clone do estilo promocaorelampagopassagens.com
   - Artigos advertorials disfarçados
   - Múltiplos ângulos de copy

3. **Mini Blog de Cases - APP/GPS**
   - Baseado em prints do app e sistema
   - Artigos curtos tipo notícia
   - Cada print vira um case de sucesso

4. **Mini Blog de Cases - Clube VIP**
   - Baseado em depoimentos do Clube de Viagens
   - Foco em cases reais de usuários

## Tecnologias
- HTML/CSS/JS estático
- Responsivo (mobile-first)
- Hospedagem: Cloudflare Pages
